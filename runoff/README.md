# runoff
The notebooks and code within this folder run the processing to create runoff datasets.

# Process
To run the runoff workflow, follow these steps:
1. Gather input datasets
2. Run `Step1-Process-RACMO.ipynb`
3. Run `Step2-CProcess-CESM_WACCM.ipynb`
4. Run `Step3-Bias-Correct-CESM_WACCM.ipynb`
5. Run `Step4-Final-Adjustments-CESM_WACCM.ipynb`
